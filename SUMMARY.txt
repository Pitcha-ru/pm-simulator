🚢 ДЕПЛОЙ НА CLOUDFLARE PAGES

Способ 1 (самый простой):
  1. npm run build
  2. Загрузите папку dist/ на Cloudflare Pages
  
Способ 2 (через Git):
  1. Запушьте код на GitHub
  2. Подключите репозиторий к Cloudflare Pages
  3. Build command: npm run build
  4. Build output: dist

Подробности: см. DEPLOYMENT.md

🎮 КАК ИГРАТЬ

1. Перетащите карточку задачи в правильный отдел
2. Успейте до истечения таймера
3. Зарабатывайте очки за скорость и точность
4. Стройте серии для множителя очков (до x1.5)
5. Проходите уровни (каждые 8 задач)
6. Доберитесь до 30 уровня!
